# uchos
This is a simple operating system written in C/C++ and Assembly.

It is a hobby project and is not intended to be used in any production environment

CPU Architecture: x86-64

## Features
- [x] Bootloader
- [x] GDT
- [x] IDT
- [ ] PIC
- [x] Keyboard
- [x] Timer
- [x] Memory Management
  - [x] Buddy Memory Allocator
  - [x] Slab Allocator
- [x] Paging
- [x] Multitasking
- [x] File System
  - [x] FAT32
- [ ] Shell
- [ ] Basic Commands
- [ ] Basic Programs
- [ ] Basic Networking
